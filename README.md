# Python Sample
- 방탈 라이브러리를 이용한 프로젝트입니다.
- 이 프로그램의 소스 코드는 방탈 게임서버 버전 0.2.0.0에서 작성되었습니다.
- 모든 프로젝트(게임) 조작은 마우스 클릭으로 이루어집니다.
- 각 프로젝트에 맞는 이미지 폴더를 사용합니다.(예 - sample1.py는 image1폴더 이미지를 사용함)
